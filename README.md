# LoyaltyCard
Cartão de fidelidade digital para cafés, padarias, salões e barbearias portuguesas.

## Funcionalidades Planeadas
- Cartão digital de carimbos (substitui o cartão de papel)
- QR code no balcão — cliente digitaliza para receber carimbo
- Recompensas personalizáveis (café grátis ao 10º, desconto, etc.)
- Painel com analytics (clientes ativos, taxa de retorno)
- Notificações push/SMS quando falta 1-2 carimbos para o prémio
- Sem app necessária — funciona via browser (PWA)

## Stack
- Next.js 16 + TypeScript + Tailwind CSS
- PostgreSQL + Prisma
- Vercel (PWA)

## Preço Sugerido
- Gratuito: 1 cartão, 50 clientes
- Pro: €10/mês — cartões ilimitados, SMS, analytics
